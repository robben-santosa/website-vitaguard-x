# 🛡️ VitaGuard X

**VitaGuard X** adalah website informasi dan monitoring kesehatan yang menampilkan berbagai fitur untuk menjaga vitalitas tubuh. Website ini dibangun dengan teknologi HTML, CSS, dan JavaScript murni (tanpa framework backend).

## 🌐 Demo Langsung

Kunjungi website: [https://username.github.io/vitaguard-x](https://username.github.io/vitaguard-x)  
*(Ganti `username` dengan username GitHub Anda setelah di-deploy)*

## ✨ Fitur Website

- 🏠 **Landing Page** – Informasi tentang manfaat VitaGuard X
- 📊 **Dashboard Sederhana** – Menampilkan status kesehatan (detak jantung, tidur, langkah)
- 📱 **Responsif** – Tampilan rapi di HP, tablet, dan laptop
- 🎨 **Desain Modern** – Menggunakan CSS Flexbox/Grid dan animasi halus
- 📞 **Kontak** – Form sederhana untuk konsultasi (simulasi, tanpa backend)
